该项目使用 [Create React App](https://github.com/facebookincubator/create-react-app)脚手架创建的。

#### 如何启动
* start
```
npm start
```
 or
```
yarn start
```
* build
```
npm run build
```
 or
```
yarn build
```

#### 项目框架

该项目使用的react+react-router+redux+redux-thunk+antd全家桶                                                
欢迎查看[docs](https://www.jianshu.com/p/9867d30afdb6)
